**EOP (Engine Overhaul Project) Starter Kit**

**What does this mod include?**

- [Engine Overhaul Project](https://github.com/EOP-Labs/M2TWEOP-library) V3.3.1 with the following features
    - Characters can now hold 16 ancillaries
    - AI armies are automatically sorted by unit type
    - Bodyguard regen limit removed
    - DaC Discord Rich Presence
    - All console commands enabled
    - Highlight units on the battlemap (CTRL + Q)
    - [Preview any battlemap](https://www.youtube.com/watch?v=RrGi4zxr7bU) from the campaign map (MMB -> Preview map)
    - Custom EOP DaC Theme
    - Batch file for automatically converting saves to EOP saves
    - Fixes for missing building UI
- DaC V5 Community Patch -> https://discord.com/channels/275599288741003265/347790917408915456/1421623408390832239

**How to install?**
Download and drag and drop the **contents** of the zip file into `Medieval II Total War/mods/Divide_and_Conquer` or equivalent mod folder.

**How to use?**
Run `M2TWEOP GUI.exe` to launch the mod. If you need to convert existing saves, double click `convert_saves_to_eop_saves.bat`

**How to use vanilla/uninstall?**
Launch via DaC_Launcher.exe/Run `Uninstall_EOP.bat`
